# 🎓 CVR College Event Management System

A responsive web-based Event Management System developed using **HTML5, CSS3, JavaScript, and Bootstrap 5**. This project helps manage technical, cultural, and club events conducted at CVR College of Engineering.

---

## 🚀 Features

### 🏠 Home Page
- Responsive Bootstrap Navbar
- Event Carousel
- Featured Upcoming Events
- Event Alerts
- Mobile-Friendly Design

### 📅 Events Page
- Technical Symposium
- Hackathon
- Coding Competition
- AI Workshop
- Cultural Fest
- Event Details Modal
- FAQ Accordion

### 📝 Registration Page
- Event Registration Form
- Input Validation
- Event Selection Dropdown
- Terms & Conditions Checkbox
- Registration Success Alert

### 🎭 Clubs Page
- Art Club
- Theatre Club
- Book Reading Club
- Club Information Cards
- Join Club Buttons

### 🖼 Gallery Page
- Event Photo Gallery
- Responsive Grid Layout
- Bootstrap Cards & Images

### 📊 Admin Dashboard
- Total Events Statistics
- Registrations Count
- Volunteers Count
- Clubs Count
- Event Records Table
- Badges
- Progress Bars
- Offcanvas Menu
